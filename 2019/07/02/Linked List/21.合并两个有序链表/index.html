方法一： 迭代

思路
  1. 定义 prev 做为新链表头，定义 res 做为结果链表
  2. 判断 l1 和 l2 的值，将 prev.next 的指针指向数值小的值
  3. 跳出循环后，说明 l1 或者 l2 中有一个链表为空，则将 prev.next 指向剩下不为空的链表

复杂度分析
  1. 时间复杂度：O(m + n)，m 和 n 分别为 l1 和 l2 的长度
  2. 空间复杂度：O(1)
